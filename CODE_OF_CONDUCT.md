# Code of Conduct

## Our Pledge

We are committed to a welcoming, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and focus on collaboration.
- Give and receive feedback professionally.

## Unacceptable Behavior

- Harassment, hate speech, personal attacks, or discrimination.
- Trolling, intimidation, or doxxing.
- Any conduct that creates an unsafe environment.

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards.
They may remove or edit comments, commits, code, or other contributions that violate this policy.

## Reporting

Please open a private security or moderation report via GitHub if possible.
If private reporting is unavailable, open an issue with minimal sensitive detail and request maintainer contact.
